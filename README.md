# ENV Configuration
```
KITE_API_KEY=<get this from inspect of kite website - enctoken>
ENV=<LOCAL for windows machine and PROD for ubuntu machine>
SYMBOL=<kite symbol>
EXCHANGE=<exchange based on symbol NSE, BSE, NFO, etc>
MONGO_DB=<dev used for testing purpose and prod used for realtime market>
MONGO_URI=<get this from MongoDB atlas>
EMAIL_ADDRESS=<email address from which email notifications should be sent>
EMAIL_PASSWORD=<password for above email address>
EMAIL_RECIPIENTS=<comma separated notification receivers email>
```
